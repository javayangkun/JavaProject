

```tex
git add --all 提交所有文件

git status 查看所有已在缓冲区的文件

git commit -m 'xxxx'  给本次提交文件加注释

git remote add origin git@gitee.com:yangkun121/java-project.git 添加远程仓库地址

git push origin master 　　 将本地仓库的文件push到远程仓库(若 push 不成功，可加 -f 进行强推操作)

git pull 拉取远程仓库文件

git checkout master 切换到master 主分支

git branch dev 创建dev分支

git merge dev 合并dev分支

git branch -a 查看所有分支包括远程分支

git beanch -d dev 删除分支dev
```
